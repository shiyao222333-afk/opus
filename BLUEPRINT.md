## 🎯 愿景

**一人即公司，个体即超级。**

一个普通人，借助 AI 工具链，完成从「信息摄入 → 知识提炼 → 真伪验证 → 内容创作 → 多平台分发 → 私域经营」的完整闭环。

如同炼金术士将铅转化为黄金——我们将互联网上的噪音转化为可盈利的知识资产。

---

## 🏛️ 命名体系：炼金术士的工坊

炼金术的核心隐喻完美契合本项目的本质：

| 阶段 | 炼金术概念 | 含义 | 对应 |
|------|-----------|------|------|
| **Nigredo** 黑化 | 分解、腐败、回归混沌 | 将原始内容分解为基本单元 | 数据摄入 |
| **Albedo** 白化 | 洗涤、净化、去除杂质 | 清洗数据、提取真知 | 知识提取 |
| **Citrinitas** 黄化 | 转化、觉醒、内在之光 | LLM 理解、结构化 | 知识转化 |
| **Rubedo** 红化 | 完成、哲人石诞生 | 最终产出、变现 | 内容输出 |

每个子项目以**炼金术工具/概念**命名，合在一起构成完整的「巨作」(Magnum Opus)。

### 🏭 Athanor（恒炉）— 知识引擎

> *Athanor 是炼金术士的自给熔炉，一旦点燃永不熄灭。*

| 属性 | 值 |
|------|-----|
| **旧名** | 知炬 / KnowledgeForge |
| **功能** | 知识库的存储、检索、管理 |
| **输入** | Markdown文档、网页文章、对话记录、视频字幕 |
| **核心** | Qdrant 向量库 + Ollama 嵌入 + LLM 问答 |
| **状态** | 🔨 已完成 MVP (Phase 3 多页面架构) |

**地位**: 整个系统的中央知识库。所有其他工具产出的知识最终流入 Athanor。

---

### ⚗️ Alembic（蒸萃）— 内容提炼

> *Alembic 是蒸馏器，将混合物加热，蒸汽上升，冷凝为纯液。*

| 属性 | 值 |
|------|-----|
| **旧名** | VideoForge / ClipForge |
| **功能** | 视频/音频 → 字幕 → 结构化文档 |
| **输入** | B站链接 → 未来: YouTube, 小红书, 播客 |
| **核心** | yt-dlp + faster-whisper + LLM 文档化 |
| **状态** | 📋 规划完成，待开发 |

**地位**: 外部信息的主要入口之一。将「别人的内容」蒸馏为「自己的知识」。

## 🛡️ 护城河（竞争壁垒）

### 1. 数据飞轮
每个子项目不是孤立的——它们互相喂养：
- Alembic 产出的文档 → 存入 Athanor 增强检索
- Crucible 的验证结果 → 提升 Athanor 内容质量
- Athanor 的知识 → 供给 Elixir 生成高质量内容
- Elixir 发布的反馈数据 → 回流优化 Alembic 的分析

竞争对手可以复制一个工具，但无法复制整个飞轮。

### 2. 个人知识图谱
系统积累的不是通用数据，而是**你**的阅读习惯、兴趣领域、验证标准。
- 你的知识库结构是独一无二的
- 你的矛盾检测标准是你亲自校准的
- 你的内容风格模型是你在使用中训练出来的

### 3. 跨源矛盾检测
目前市面上没有任何产品能做到「跨B站视频 + 个人知识库 + 网页文章」的统一矛盾检测。这是一个技术壁垒。

### 4. 端到端闭环
从摄入到变现是完整的，不是拼凑的工具链。
- 市面上的替代方案：Notion + ChatGPT + 手动下载 + 手动分析
- Opus：一条自动化管道

### 5. 开源但不可复制
- 代码开源供参考
- 但具体配置、Prompt模板、个人知识库、调优参数是私有的
- 架构理解不难，积累到同等量级需要时间

### 6. 命名与品牌壁垒
炼金术命名体系天然具有辨识度和故事性：
- 搜索 "Athanor" 不会被淹没在通用词中
- 五器工坊的概念可以注册商标
- Logo 体系可以围绕炼金术符号构建（🜂 🜃 🜁 🜄 等 Unicode 符号）
