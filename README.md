**Opus Magnum**（巨作）是我个人 AI 工具链的总体规划仓库。

| 项目 | Emoji | 功能 | 状态 |

| ⚗️ **Alembic** 蒸萃 | `⚗️` | 视频→知识提炼 | 📋 规划 |
